## 游戏操作
---
#### 用编辑器打开 SI00B_Project 后运行 `main.py`，或直接通过在 SI00B_Project 目录打开终端并输入 `python main.py` 运行

#### 按下E与NPC交互

#### 按下WASD控制玩家移动

#### 按下J攻击

#### 按下K身后斩
#### 按下SPACE翻滚

#### 按下esc退出对话

#### 杀死房间内的所有敌人门会打开，篝火会点亮
#### 靠近开启的门和点亮的篝火进行交互
####  作弊操作：在篝火升级和继续冒险的选项中选择3（隐藏）
